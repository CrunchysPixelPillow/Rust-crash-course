## editers for rust

Zed (Zed Industries | zed.dev)
  * Best for Memory-usage
  * Built in rust extention
  * Unideal for Beginners

RustRover (Jetbrains | jetbrains.com/rust)
  * Best for Beginners
  * Good for fast setup
  * Built in cargo-runtime (less work)
  * Bad for Memory-usage Around 4Gb

VScode (Microsoft | code.visualstudio.com)
  * Best for existing VScode-users
  * Better then rustrover Memory-usage around 2Gb
  * Annoying setup long-term

Overall
Beginners - RustRover
Rust Enjoyers - Zed
VS-code users - Vscode
